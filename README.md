Noolava is a modern, lightweight web application designed to connect traditional handloom weavers across major Tamil Nadu textile hubs directly with consumers. Built using a zero-dependency frontend stack, the platform prioritizes rapid performance over low-bandwidth mobile networks and extreme user-interface accessibility.

## ⚡ UX & Accessibility Engineering (TN Localized)

Standard e-commerce templates don't cut it for traditional weavers or local tier-2/tier-3 buyers. Noolava relies on a specialized **Cognitive-Load-Zero** layout system:

*   **🎙️ Integrated Voice Operations:** Built-in web speech processing framework inside the main search terminal. Users can tap the mic to speak conversational Tamil (e.g., *"Salem cotton vesti"*) to fetch direct matches.
*   **🌐 Bilingual Interface Toggle:** Flawless runtime switching between English and localized everyday Tamil phrasing to ensure users of any technical background can navigate the app comfortably.
*   **🛡️ Heritage Verification Stamps:** Transparent UI badges highlighting Government Geographical Indication (GI) verification status and official Handloom Marks directly on product listings to combat powerloom counterfeits.
*   **💬 Conversational Checkouts:** A friction-free single-tap API shortcut that routes buyer inquiries straight to the weaver's WhatsApp, leveraging the natural messaging habits already trusted in rural commerce.

---

## 🛠️ Architecture & Tech Stack

The application is engineered for pure client-side efficiency, maximizing speed on budget mobile devices and desktop browsers alike.

*   **Structure:** Semantic HTML5 providing accessible document object models (DOM) across screen readers.
*   **Styling System:** Modern, high-fidelity responsive CSS3 grid and flexbox layouts emphasizing warm neutrals, deep zari textures, and high contrast buttons.
*   **Application Logic:** Vanilla JavaScript managing product state filtering, interactive modal routing, and payment/chat API handlers.
